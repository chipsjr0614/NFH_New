급성 심근경색 (STEMI)

"가슴이 쥐어짜인다" — 응급실이 뒤집어지는 그 순간

Door-to-Balloon 90분 12-lead ECG DAPT PCI ST 상승
📚 지식 교육 섹션

이 병을 이해하자

정의 · 왜 생기는지 · 왜 이 증상인지 · 왜 이 치료인지

관상동맥이 완전히 막혀 심근에 혈류가 완전 차단되고, 심전도에서 ST분절 상승이 나타나는 급성 심근경색.
골든타임(90분) 내에 재관류하지 않으면 심근세포가 영구적으로 괴사한다.

NSTEMI와 차이: NSTEMI는 부분 폐색으로 ST 상승 없이 Troponin만 상승. STEMI는 완전 폐색으로 훨씬 더 위험하고 즉각적인 PCI가 필요하다.

🔬 발생 메커니즘 — 왜 생기나요?
1
죽상경화증 (Atherosclerosis) 형성
LDL 콜레스테롤이 혈관 내피세포 손상 부위에 침착 → 대식세포가 LDL을 먹어 거품세포(foam cell) 형성 → 지질 코어를 가진 플라크가 혈관벽에 수년~수십 년 쌓임
2
플라크 파열 (Plaque Rupture)
불안정한 플라크의 얇은 섬유성 막(fibrous cap)이 파열 → 지질 코어 내부의 조직인자(tissue factor)가 혈액에 노출
3
혈소판 응집 & 혈전 형성
노출된 조직인자 → 혈소판 활성화 → 응고 연쇄반응 → 혈전(clot) 형성 → 관상동맥 완전 폐색
4
심근 허혈 → 괴사
폐색 후 20분부터 심근세포 괴사 시작 → 6시간 내 재관류 없으면 전층 심근경색 완성

🧠 왜 흉통이 생기는가? 심근에 산소 공급이 차단 → 무산소 대사로 젖산 축적 + 칼륨·브래디키닌 방출 → 통증 수용체 자극.
방사통의 이유: 심장 통증 신호가 C7~T4 척수 분절 → 같은 분절의 좌측 팔·턱·어깨로 오인 (referred pain).

🩺 증상과 기전 — 왜 이 증상이 나타나나요?
증상기전 (왜 생기나요?)
흉통 · 압박감
30분 이상, 쥐어짜는 느낌		 무산소 대사 → 젖산·브래디키닌 축적 → 심근 통증 수용체 자극

⚠️ 비전형 증상 — 놓치기 쉬운 경우

• 당뇨 환자: 신경병증으로 통증 못 느껴 → 무통성 심근경색
• 노인/여성: 오심·구토·피로감만 호소
• 명치 통증: 소화불량으로 오인 → 의심되면 ECG 먼저

💊 치료 — 왜 이 치료를 하나요?
핵심 원칙 STEMI 치료의 핵심 = 최대한 빨리 막힌 혈관을 다시 열기 (재관류)
모든 초기 처치는 재관류 전까지 심근 손상을 최소화하기 위함이다.
1순위Primary PCI (경피적 관상동맥 중재술)
왜? 막힌 관상동맥에 직접 스텐트 삽입 → 재관류. D2B 90분 이내 목표.
약물Aspirin 300mg (씹어서)
왜? COX-1 비가역적 억제 → TXA2 생성 차단 → 혈소판 응집 억제. 씹으면 흡수 2~3배 빠름.
약물P2Y12 억제제 (Ticagrelor / Clopidogrel)
왜? ADP 수용체 차단 → Aspirin과 다른 경로로 혈소판 추가 억제 (이중 항혈소판 = DAPT).
약물Heparin (항응고제)
왜? 응고인자 Xa·IIa 억제 → 혈전 확장 방지 + PCI 시 카테터 혈전 방지.
산소산소 투여 (SpO₂ <90%인 경우만)
왜 조건부인가? SpO₂ 정상인 경우 과산소혈증이 심근 손상 증가시킬 수 있음 (최신 가이드라인).
약물Nitroglycerin (NTG)
왜? NO 방출 → 관상동맥·정맥 확장 → 전부하 감소.
금기: SBP <90, 우심실 경색, 24시간 내 PDE5 억제제 복용.
🚨 하벽 STEMI (RCA 폐색) 특별 주의

RV infarction 동반 가능 → 전부하(preload) 의존적.
NTG, 이뇨제 → 절대 금기 (급성 저혈압 위험)
치료: 수액 충분히 → 전부하 유지

⚡ 합병증 — 시간대별 뭘 조심해야 하나요?
시간대합병증기전 / 징후
즉시~수시간 심실세동 (VF) 허혈 심근의 전기 불안정 → 가장 흔한 사망 원인
24~72시간 심인성 쇼크 대면적 경색 → 좌심실 기능 급감 → 저혈압·조직관류 저하
2~7일 심실 파열 괴사 심근 연화 → 파열 → 즉각 심장눌림증, 드물지만 치명적
수일~수주 Dressler 증후군 괴사 심근 항원 → 자가면역 반응 → 발열·흉통·심낭 마찰음
수주 이후 만성 심부전 경색 부위 섬유화 → 심실 리모델링 → 만성 심부전

⚡ 지식 섹션 핵심 암기

  • STEMI = 관상동맥 완전 폐색 → ST 상승 → 골든타임 90분
  • 플라크 파열 → 혈전 → 폐색 → 20분부터 괴사 시작
  • LAD → 전벽 (V1–V4), RCA → 하벽 (II·III·aVF), LCx → 측벽 (I·aVL·V5–V6)
  • 하벽 STEMI → RV infarction 가능 → NTG 금기, 수액 줘야 함
  • 산소는 SpO₂ <90%일 때만
  • Aspirin은 씹어서 (흡수 2~3배 빠름)

⚠️ 주의!

BP 90 이하 → Nitroglycerin 절대 금기! 혈압이 더 떨어져 쇼크 악화. 등이 찢어지는 통증이면 대동맥 박리 감별 먼저 (Heparin 금기 달라짐).

🚨 절대 놓치면 안 되는 신호
  • 🔴 ECG에서 ST 상승 2mm 이상 → 즉시 의사 호출, PCI 준비
  • 🔴 새로운 LBBB 출현 → STEMI와 동등하게 처치
  • 🔴 혈압 90 이하 → Cardiogenic shock, 승압제 준비
  • 🔴 갑자기 의식 저하 → CPR 준비, 제세동기 켜두기

❓ 케이스 스터디 퀴즈
절대 금지. BP 90 이하에서 NTG 투여 시 혈압 추가 하락 → 쇼크 악화. 수액 소량 투여 + 승압제(Dopamine/Norepinephrine) 고려 후 즉시 의사 보고.

즉시 Code Blue + CPR + 제세동. 1. "도와주세요!" + Code 콜 2. CPR 시작 (100-120회/분, 5cm↑) 3. 제세동기 → 리듬 확인 4. VF/무맥 VT → 200J 제세동 5. 제세동 직후 CPR 재시작 2분 6. 2주기 후 Epinephrine 1mg IV
