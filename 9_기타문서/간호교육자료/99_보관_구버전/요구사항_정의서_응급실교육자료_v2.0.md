## 1. 프로젝트 개요

응급실에서 자주 만나는 내과·외과·정형·흉부외과·산부인과·소아과 질환을
케이스 기반으로 공부하고, 스스로 발표하며 성장하는 **신규 의료인 자기주도 학습 시스템**

### 핵심 변경사항 (v1.0 → v2.0)

v1.0은 케이스(임상 실전) 중심이었다면,  
v2.0은 케이스 앞에 **지식 교육 섹션**을 추가하여 이론 → 실전 흐름으로 구성

```
[v1.0 구조]
케이스 오프닝 → 선배 해설 → 처치 순서 → 팁 → 퀴즈

[v2.0 구조]
📚 지식 교육 섹션 (신규 추가)
        ↓
🚨 케이스 오프닝 → 선배 해설 → 처치 순서 → 팁 → 퀴즈
```

## 3. 교육 철학

### 3-1. 프셉마음 스타일 (핵심 원칙)

모든 교육 섹션은 **선배 간호사가 신규에게 옆에서 말로 설명해주는 톤**으로 작성

```
원칙
─────────────────────────────────────────────────────
✅ "이거 이렇게 하는 거야, 왜냐하면..."
✅ 어려운 의학용어 → 일상 비유로 먼저 설명
✅ 신규가 자주 하는 실수 미리 알려주기
✅ "나중에 배워도 돼 / 일단 이것만" 우선순위 표현
✅ 말풍선·구어체 박스로 선배 코멘트 시각화

❌ 딱딱한 교과서 문체
❌ 이유 없이 "이렇게 해라"만 나열
❌ 신규가 이미 안다고 가정하는 표현
```

예시:
> "ECG 처음 보면 뭐가 뭔지 모르지? 일단 ST만 봐. 기준선보다 올라갔으면 즉시 보고야. 나머지는 나중에 배워도 돼."

### 3-2. 이론적 근거 필수 포함 원칙

**모든 검사·처치·약물에는 반드시 "왜 하는지" 이론적 근거를 함께 제시**

```
형식 예시
─────────────────────────────────────────────────────
약물명: Aspirin 300mg — 씹어서 투여
근거: COX-1 억제 → TXA₂ 차단 → 혈소판 응집 억제
      씹으면 흡수 3분 vs 삼키면 15분 (속도 차이)

검사명: Serial Troponin (6시간 후 재검)
근거: 심근세포 괴사 → 세포막 파괴 → Troponin 혈액 유출
      초기 음성 가능 → 6시간 경과 후 유의한 상승 확인
```

## 6. AMI 교육 콘텐츠 상세 요구사항

### 6-1. 병태생리 (탭 ②)

```
정상 혈관 → 죽상경화반 형성 (수년) → 플라크 파열 (갑자기)
→ 혈전 형성 → 부분 폐색(NSTEMI) / 완전 폐색(STEMI)
→ 심근 허혈 → 괴사 (20분 시작 / 6시간 광범위)
```

- DAPT가 혈전 형성 단계를 막는다는 연결고리 명시
- "수도관" 비유 텍스트 포함 (이미지로는 실제 단면도 사용)

### 6-2. STEMI vs NSTEMI 비교표

| 구분 | STEMI | NSTEMI |
|---|---|---|
| 폐색 정도 | 완전 폐색 | 부분 폐색 |
| ECG | ST 상승 | ST 하강 또는 정상 |
| 심근효소 | 상승 | 상승 |
| 치료 목표 | D2B 90분 PCI | 안정화 후 PCI |

### 6-3. 관상동맥 지도 (탭 ③)

| 혈관 | 공급 부위 | 막히면 ECG | 특별 주의 |
|---|---|---|---|
| LAD | 심실 전벽 | V1~V4 ST↑ | 가장 위험 |
| RCA | 하벽·우심실 | II·III·aVF ST↑ | NTG 금기, 수액 |
| LCX | 측벽 | I·aVL·V5~V6 ST↑ | 진단 어려움 |

### 6-4. 증상별 이론적 근거 (탭 ④)

| 증상 | 이론적 근거 |
|---|---|
| 흉통·압박감 | 허혈 심근 → 젖산·브레디키닌 축적 → 통증 수용체 자극 |
| 왼팔·턱 방사통 | 심장 구심성 신경(C8~T4) = 왼팔·턱 피부분절 같은 척수 레벨 → 연관통 |
| 식은땀·창백 | 심박출량↓ → 교감신경 활성 → 카테콜아민 → 말초혈관 수축 |
| 오심·구토 | RCA 경색 → 하벽 자극 → 미주신경 반사 항진 |
| 호흡곤란 | 좌심실 기능↓ → 폐정맥 압력↑ → 폐부종 |

### 6-5. ECG (탭 ⑤)

**이론적 근거:**
심근 허혈 → 세포막 이온 펌프 손상 → 재분극 이상 → ST분절·T파 변화
어느 유도에서 변화 → 어느 혈관이 막혔는지 파악 가능

**ECG 변화 순서:**
과첨예 T파(초기) → ST 상승★ → T파 역전 → 병적 Q파(괴사 완성)

**진단 기준:** 인접 2개 유도에서 1mm↑ (V2-V3은 2mm↑)

**Serial Troponin 근거:**
심근세포 괴사 → 세포막 파괴 → Troponin 혈액 유출
초기 음성 가능 → 6시간 후 재검 필수

### 6-6. 치료별 이론적 근거 (탭 ⑥)

| 약물·처치 | 이론적 근거 |
|---|---|
| Aspirin 300mg (씹어서) | COX-1 억제 → TXA₂ 차단 → 혈소판 응집 억제. 씹으면 3분 흡수 |
| Clopidogrel | ADP 수용체(P2Y12) 차단 → Aspirin과 다른 경로 → DAPT = 이중 차단 |
| 산소 (SpO₂<90%만) | 정상 SpO₂에서 과산소 → 활성산소↑ → 심근 손상 악화 |
| NTG | 혈관 확장 → 전부하↓ → 심근 산소 요구량↓ |
| Morphine | 통증·교감 흥분 완화 → 심근 산소 소비↓ |
| Primary PCI | 혈전 직접 제거 + 스텐트 고정. D2B 90분 목표 |

**NTG 금기 2가지:**
1. SBP < 90mmHg → 혈압 더 떨어져 쇼크
2. 하벽 STEMI(RCA) → 우심실 경색 동반 → 전부하 의존 → 수액 줘야

## 7. 전체 케이스 표준 구성 (8단계 — v1.0에서 유지)

```
[지식 교육 섹션] ← v2.0 신규 추가 (6개 탭)
        ↓  구분선
1. 🚨 케이스 오프닝  — 119 무전·시간·바이탈, 드라마처럼 시작
2. 💭 스스로 생각하기 — "지금 뭘 해야 할까?" 클릭하면 정답 공개
3. 🧠 선배 해설      — 말풍선 스타일 대화체
4. ✅ 지금 당장 할 것 — 행동·이유·주의사항 테이블
5. 💬 현장 팁        — 교과서에 없는 실전 지식
6. 🔴 Red Flag       — 절대 놓치면 안 되는 신호
7. ⚡ 핵심 암기 박스
8. ❓ 케이스 스터디 퀴즈 — 스포일러 방식
```
