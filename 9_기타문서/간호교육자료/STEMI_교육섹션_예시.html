❤️ 심장/순환기 — Case 01

급성 심근경색 (STEMI)

ST 분절 상승형 심근경색 · 교육 섹션 예시

📚 NEW · 지식 교육 섹션 (추가 예정)
01 · 정의
📖 이 병이 뭔가요?
STEMI 정의 관상동맥이 완전히 막혀 심근에 혈류가 완전 차단되고, 심전도에서 ST분절 상승이 나타나는 급성 심근경색. 골든타임(90분) 내에 재관류하지 않으면 심근세포가 영구적으로 괴사한다.

NSTEMI와 차이: NSTEMI는 부분 폐색으로 ST 상승 없이 Troponin만 상승. STEMI는 완전 폐색으로 훨씬 더 위험하고 즉각적인 PCI가 필요하다.

완전 폐색 ST 상승 Troponin 상승 골든타임 90분 즉각 PCI 필요
02 · 위험인자
⚠️ 누가 잘 걸리나요?

죽상경화증이 수년~수십 년에 걸쳐 쌓이다가 어느 순간 플라크가 파열하면서 STEMI가 발생한다.

조절 가능한 위험인자
고혈압 당뇨병 고지혈증 흡연 비만 운동 부족 스트레스
조절 불가능한 위험인자
고령 (남 ≥45세, 여 ≥55세) 남성 가족력
03 · 병태생리 — 왜 생기나요?
🔬 발생 메커니즘
1
죽상경화증 (Atherosclerosis) 형성
LDL 콜레스테롤이 혈관 내피세포 손상 부위에 침착 → 대식세포가 LDL을 먹어 거품세포(foam cell) 형성 → 지질 코어(lipid core)를 가진 플라크가 혈관벽에 쌓임
2
플라크 파열 (Plaque Rupture)
불안정한 플라크의 얇은 섬유성 막(fibrous cap)이 파열 → 지질 코어 내부의 조직인자(tissue factor)가 혈액에 노출
3
혈소판 응집 & 혈전 형성 (Thrombosis)
노출된 조직인자 → 혈소판 활성화 → 응고 연쇄반응 → 혈전(clot) 형성 → 관상동맥 완전 폐색
4
심근 허혈 → 괴사
폐색 후 20분부터 심근세포 괴사 시작 → 6시간 내 관류 회복 없으면 전층(transmural) 심근경색 완성
관상동맥 내 죽상동맥경화 플라크 단면도 — 지질 코어와 섬유성 막 구조

🧠 왜 흉통이 생기는가?

심근에 산소 공급이 차단되면 무산소 대사로 젖산(lactic acid)이 축적되고, 심근 손상으로 칼륨·브래디키닌·프로스타글란딘 같은 통증 매개물질이 방출된다. 이것이 교감신경 → 척수 → 시상 경로를 거쳐 흉통으로 인식된다.

방사통의 이유: 심장의 통증 신호가 C7~T4 척수 분절로 올라가는데, 같은 분절을 쓰는 팔(특히 좌측), 턱, 어깨가 아프게 느껴진다 (referred pain).

05 · 임상 증상 — 왜 이 증상이 나타나나요?
🩺 증상과 기전
증상 기전 (왜 생기나요?)
흉통/압박감
심한 압박, 쥐어짜는 느낌
30분 이상 지속
 무산소 대사 → 젖산·브래디키닌 축적 → 심근 통증 수용체 자극
좌측 팔/턱 방사통
어깨, 목, 명치 방사 가능

식은땀 / 창백
 심박출량 감소 → 교감신경 활성화 → 말초혈관 수축, 땀샘 자극
오심 / 구토
 하벽 MI (RCA 폐색) → 미주신경 자극 → 서맥 + 오심/구토
호흡곤란
 좌심실 기능 저하 → 폐울혈 → 가스교환 저하
실신 / 의식 변화
심한 경우
 심박출량 급감 or 심한 부정맥 → 뇌관류 저하
⚠️ 비전형 증상 — 놓치기 쉬운 경우
  • 당뇨 환자: 무통성 심근경색 (신경병증으로 통증 못 느낌)
  • 노인/여성: 오심·구토·피로감만 호소하는 경우 많음
  • 명치 통증: 소화불량으로 오인 주의 → 의심되면 ECG 먼저
06 · 진단 — ECG가 왜 그렇게 변하나요?
📈 STEMI ECG 변화의 이유
초기 (수분~1시간)
과첨예 T파 (Hyperacute T)
세포 내 칼륨 이동 → 재분극 변화. 발견하기 어렵지만 가장 초기 징후
급성기 (수분~수시간)
ST 분절 상승 ▲
허혈 심근의 손상 전류(current of injury) → 손상된 영역 상방에서 ST 상승
수시간~수일
T파 역전
심근 재분극 이상 → 양성 T파가 음성으로 반전 (경색 확인 증거)
수일~수주 후 (영구)
병적 Q파
괴사 심근은 전기 신호 없음 → 반대 방향 탈분극만 기록 → Q파 형성
전벽 STEMI 실제 12유도 ECG — V1–V4 ST 상승, 상호 변화 확인
© Cureus Journal / Wikimedia Commons (CC BY 4.0)
📋 STEMI 진단 기준 (ST 상승 기준값)
  • V2–V3: 남성 <40세 ≥2.5mm, 남성 ≥40세 ≥2mm, 여성 ≥1.5mm
  • 다른 유도: 인접 2개 이상 유도에서 ≥1mm
  • 후벽 MI 의심: V7–V9 유도 확인 (LCx 폐색)
07 · 치료 — 왜 이 치료를 하나요?
💊 치료 원칙과 기전
STEMI 치료의 핵심 = 최대한 빨리 막힌 혈관을 다시 열기 (재관류)
모든 초기 처치는 재관류 전까지 심근 손상을 최소화하기 위함이다.
1순위 Primary PCI (경피적 관상동맥 중재술)
왜? 막힌 관상동맥에 직접 스텐트를 삽입해 재관류. 혈전용해제보다 성공률 높고 출혈 합병증 적음. D2B(Door-to-Balloon) 시간 90분 이내 목표.
약물 Aspirin 300mg (씹어서)
왜? COX-1 비가역적 억제 → TXA2 생성 차단 → 혈소판 응집 억제. 씹어야 하는 이유: 분쇄 시 흡수 속도가 2–3배 빠름.
약물 P2Y12 억제제 (Ticagrelor or Clopidogrel)
왜? ADP 수용체 차단 → 혈소판 활성화 추가 억제 (Aspirin과 다른 경로). 이중 항혈소판요법(DAPT)으로 혈전 재형성 방지.
약물 Heparin (항응고제)
왜? 응고인자 Xa·IIa 억제 → 혈전 확장 방지 + PCI 시 카테터 혈전 방지. PCI 전까지 IV 투여.
산소 산소 투여 (SpO2 <90%인 경우만)
왜 조건부인가? 과거 모든 환자에게 줬지만, 최근 연구에서 SpO2 정상인 경우 과산소혈증이 오히려 심근 손상을 증가시킬 수 있음 확인. 현재 가이드라인: SpO2 <90% 또는 호흡곤란 있을 때만 투여.
약물 Nitroglycerin (NTG)
왜? NO 방출 → 관상동맥·정맥 확장 → 전부하 감소 + 허혈 경감.
금기: 수축기혈압 <90mmHg, 우심실 경색(RCA 폐색), 24시간 내 PDE5 억제제(Sildenafil 등) 복용 시 → 심한 저혈압 유발.
약물 Morphine (필요 시)
왜? 통증·불안 경감 → 교감신경 활성 감소 → 심박수·혈압·산소 소모 감소.
주의: 항혈소판제 흡수 저해 가능성 → 꼭 필요한 경우에만.
🚨 RCA 폐색(하벽 STEMI) 특별 주의사항
  • RV infarction 동반 가능 → 전부하(preload) 의존성 증가
  • NTG, Diuretics → 절대 금기 (급성 저혈압 위험)
  • 치료: 수액 충분히 → 전부하 유지
  • 서맥·저혈압·경정맥 확장 → Bezold-Jarisch 반사 의심
08 · 합병증 — 뭘 조심해야 하나요?
⚡ 주요 합병증과 시간대
시간대 합병증 기전 / 징후
즉시~수시간 심실세동 (VF) 허혈 심근의 전기적 불안정 → 가장 흔한 사망 원인, 제세동 준비
24~72시간 심인성 쇼크 대면적 경색 → 좌심실 기능 급감 → 저혈압·조직관류 저하
2~7일 심실 파열 괴사 심근 연화 → 파열 → 즉각 심장눌림증, 드물지만 치명적
수일~수주 심낭염 (Dressler 증후군) 괴사 심근 항원 → 자가면역 반응 → 발열·흉통·심낭 마찰음
수주 이후 심부전 / 심실류 경색 부위 섬유화 → 심실 리모델링 → 만성 심부전

⚡ 핵심 암기 — STEMI 교육 섹션

  • STEMI = 관상동맥 완전 폐색 → ST 상승 → 골든타임 90분
  • 플라크 파열 → 혈전 → 완전 폐색 → 허혈 → 20분부터 괴사 시작
  • LAD 막히면 전벽 (V1–V4), RCA 막히면 하벽 (II·III·aVF), LCx 막히면 측벽 (I·aVL·V5–V6)
  • 하벽 STEMI → RV infarction 가능 → NTG 금기, 수액 줘야 함
  • 산소는 SpO2 <90%일 때만 (정상이면 오히려 해로울 수 있음)
  • Aspirin은 씹어서 (흡수 2–3배 빠름)
  • D2B 90분 목표 — 병원 도착 즉시 카드팀 Call
