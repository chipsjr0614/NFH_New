신규 응급실 의료인을 위한 케이스 기반 자기주도 학습 플랫폼

📅 작성일: 2026-06-12 📌 버전: v1.0 🎯 대상: 신규 간호사 · 신규 응급구조사
1

프로젝트 개요

프로젝트 한줄 정의 응급실에서 자주 만나는 내과·외과·정형·흉부외과·산부인과·소아과 질환을 케이스 기반으로 공부하고, 스스로 발표하며 성장하는 신규 의료인 자기주도 학습 시스템
교육 철학 "프셉마음" 스타일 — 선배 의료인이 옆에서 말로 설명해주는 것처럼, 딱딱하지 않고 실전적이며 친근하게. 두꺼운 교과서가 아니라, 현장에서 바로 써먹는 살아있는 지식.
핵심 차별점 단순 지식 전달 ❌ → 케이스로 생각하게 만들기 ✅ | 외우기 ❌ → 이해하고 적용하기 ✅ | 교사 주도 ❌ → 자기주도 ✅

3

교육자료 구성 형식 (케이스 1개 = 표준 템플릿)

원칙 모든 케이스는 동일한 구조로 만든다 — 익숙해지면 어디서 뭘 찾아야 하는지 바로 안다.
🚨

① 케이스 오프닝 — 드라마처럼 시작

119 무전 소리, 환자 도착 장면, 바이탈 수치로 시작. 읽는 순간 현장에 있는 느낌. "자, 지금 이 환자 앞에 네가 서 있어."

🧠

② 프리셉터 해설 — 말풍선 스타일

"이 환자 보는 순간 뭐가 떠올라야 해?" 형식의 선배 목소리. 딱딱한 설명 대신 대화체로. 왜 이 판단을 하는지 사고과정 공유.

✅

③ 지금 당장 해야 할 것 — 순서대로 테이블

행동 → 이유 → 주의사항 3열 테이블. 우선순위 명확하게. 헷갈리지 않도록 번호 순서.

💬

④ 꿀팁 / 임상 팁 박스

교과서에 없는 현장 지식. "이거 모르는 신규 많아" 형식. 암기법, 자주 하는 실수, 주의사항 포함.

🔴

⑤ 절대 놓치면 안 되는 신호 — Red Flag

즉시 행동해야 하는 위험 신호 3~5가지. 짧고 굵게. 눈에 확 들어오는 디자인.

🎯

⑥ 핵심 암기법 / 요약 박스

각 질환의 핵심을 한 줄로. 두문자어, 숫자 암기법 활용. "이것만 기억해도 된다" 박스.

❓

⑦ 케이스 스터디 퀴즈 — 접어두기 (Spoiler 방식)

3~5개 질문. 답을 숨겨두고 클릭하면 펼쳐짐. 스스로 먼저 생각하게. 발표 준비용으로도 활용.

📊

⑧ 관련 질환 링크 / 다음 케이스 안내

연계 학습 유도. 비슷한 증상의 다른 질환 연결. "이것도 같이 공부하면 좋아".

학습 활용 방법

01

자기 주도 학습

개인이 케이스 선택 후 혼자 읽기 → 퀴즈 풀기 → 모르는 것 체크

02

스터디 발표

발표 모드로 전환 → 케이스 오프닝부터 순서대로 발표 → 청중 퀴즈

03

케이스 스터디

실제 환자 케이스와 교육자료 비교 → 어디서 달랐는지 토론

04

스케줄 관리

주 1~2개 케이스 자율 배정 → 완료 체크 → 64개 완주 목표

추천 학습 순서 심정지(#06) → 패혈증(#50) → STEMI(#01) → 뇌졸중(#20) → 저혈당(#46) → DKA(#44) → 아나필락시스(#59)
→ 빈도 높고 시간이 생명인 케이스부터 먼저 학습 권장
