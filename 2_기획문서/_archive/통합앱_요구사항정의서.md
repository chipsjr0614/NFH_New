**버전:** v2.0

**폴더:** `/NFH_New/nfh_v3/`  
**파일:** `index.html` (단일 파일, 오프라인, 전체 내장)

## 1. 프로젝트 개요

국립소방병원 응급실 간호사를 위한 **통합 응급 플랫폼**.  
ER Mate v2 폼·디자인을 그대로 유지하면서  
성인약물계산기·소아약물계산기·약물정보검색·간호기록을  
메인 화면에서 바로 클릭으로 접근 가능하도록 통합.

## 7. 오버레이 시스템

### 공통 규격
| 항목 | 내용 |
|------|------|
| 스타일 | v2 헤더(`#1B3A4B`) + `← 닫기` 버튼 |
| 표시 방식 | 전체화면 슬라이드 업 (아래 → 위) |
| 배경 | 반투명 딤 처리 |
| 닫기 | `← 닫기` 버튼 또는 배경 탭 |

### 오버레이 A — 💉 성인 계산기
```
← 닫기  💉 성인 응급약물 계산기
────────────────────────────────
체중 [ 70 ] kg  [50][60][70][80][90]
────────────────────────────────
🟢 Norepinephrine
  말초 16mcg/ml  │  중심 40mcg/ml
   4mcg → 15.0  │   4mcg →  6.0
   8mcg → 30.0◀ │   8mcg → 12.0◀
  12mcg → 45.0  │  12mcg → 18.0
────────────────────────────────
🔵 Dopamine / 🟣 Dobutamine
🔴 Epinephrine / 🟡 Vasopressin
```

### 오버레이 B — 👶 소아 계산기
```
← 닫기  👶 소아 응급 계산기
────────────────────────────────
체중 [ 15 ] kg
[신생아][1세][3세][5세][7세][10세]
────────────────────────────────
[🚨 응급]  [💊 약물]  [💧 수액]
────────────────────────────────
🚨 응급 탭:
  활력징후 / 기도장비 / 인공호흡기
  제세동 / 심정지 / 경련 / 아나필락시스
💊 약물 탭:
  RSI / 진통제 / 진정제 / 항구토제
💧 수액 탭:
  볼루스 / 유지수액 / 쇼크별
```

### 오버레이 C — 🔍 약물정보 검색
```
← 닫기  🔍 약물 정보 검색
────────────────────────────────
🔍 [ 약물명 검색... ]
────────────────────────────────
카드: 농도·희석·투여시간·주의사항·안정시간
[💉 성인 계산기로 바로 계산 →]
```

### 오버레이 D — 📝 간호기록
```
← 닫기  📝 간호기록
────────────────────────────────
STEP 1 — 침습적 도관
STEP 2 — 주증상 선택
STEP 3 — 분기 질문
STEP 4 — 사정·교육·계획
STEP 5 — 의식/낙상
[간호기록 팝업]
```
