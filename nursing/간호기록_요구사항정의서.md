## 2. 전체 흐름 (5단계)

```
[STEP 1] 침습적 도관 확인
         ↓
[STEP 2] 주증상 선택 (복수 선택 가능)
         ↓
[STEP 3] 분기 질문 (증상별 소분류)
         ↓
[STEP 4] 사정·교육·계획 (한 화면, 증상당 반복)
         ── 사정 【A&E】 간호진술문
         ── 📢 환자 설명 (체크 시 P&E 자동 연동)
         ── 계획 및 평가 【P&E】 간호진술문 (항상 표시)
         ↓
[STEP 5] 의식/낙상 교육 (항상 포함)
         ↓
[완료]   간호기록 - 간호일지 팝업
```

## 6. STEP 4 — 사정·교육·계획 (핵심)

### 6-1. 화면 구조 (한 화면에 세 섹션)

```
사정 【A&E】 간호진술문      ← 파란색 헤더
  💬 질문 스크립트 + 탭 버튼
  (슬래시 → 두 버튼 / 콜론 → 텍스트 입력 / 일반 → 체크박스)

📢 환자 설명                 ← 낙상교육 스타일
  말할 문장 표시 → 체크 시 P&E 기록 자동 연동

계획 및 평가 【P&E】 간호진술문  ← 노란색 헤더
  체크박스 (간호기록_수정본 원문, 항상 표시)
  교육카드 ↔ P&E 체크박스 양방향 자동 동기화
```

### 6-2. 슬래시 항목 처리 규칙

슬래시(/)로 구분된 항목은 두 개 이상의 버튼으로 분리, **선택한 값은 완전한 문장으로 기록**

| 원본 텍스트 | 선택 | 기록되는 진술문 |
|------------|------|----------------|
| 혈변 있음/없음 | 없음 | **혈변 없음** |
| grasp power 강함/약함 | 약함 | **grasp power 약함** |
| 빛 반사 확인결과 prompt/sluggish/hippus 함 | sluggish | **빛 반사 확인결과 sluggish 함** |
| 오심, 구토 없음/오심, 구토 있음 | 있음 | **오심, 구토 있음** (양쪽 완전한 문장이면 그대로) |

> **핵심 원칙**: 선택한 버튼 텍스트만이 아니라, 주어를 포함한 완전한 간호 진술문이 기록되어야 함

### 6-3. 통증 전용 — NRS 평가 도구 (비고란)

통증 있음 선택 시 자동 펼침. **NRS 결과는 간호기록 A&E에 포함되지 않으며 팝업에서 별도 "비고란"으로 표시**

| 평가 항목 | 입력 방식 | 팝업 비고란 표시 예시 |
|-----------|-----------|----------------------|
| 통증강도 | NRS 0~10 탭 버튼 | 통증강도 7점 |
| 통증부위 | [상복부][하복부][우하복부][좌하복부][흉부][머리][전신][기타] | 통증부위 하복부 |
| 통증양상 | [쥐어짜듯][찌르듯][둔하게][타는듯][콕콕] | 통증양상 쥐어짜듯 |
| 발생시기 | [방금 전][몇 시간 전][어제][며칠 전] | 통증기간 몇 시간 전부터 |
| 지속여부 | [지속적으로][간헐적으로][지금은 없음] | 통증지속 지속적으로 |

> 모든 통증 증상(복부/흉부/두통/기타/출혈동반통증 등)에 동일 적용

### 6-4. 📢 환자 설명 스크립트 (증상별) — 안내용, 팝업 미포함

화면에 말할 문장 표시 → 체크 시 대응 P&E 항목이 자동 선택됨 (양방향 동기화)

**통증:**
- "많이 아프세요? 진통제가 필요하실 만큼 아프신가요?" → 통증을 인정해줌
- "의사 선생님께 확인받고 필요하시면 진통제 처방받아 드릴게요." → 통증 양상을 확인함
- "참으실 수 있으면 조금 지켜보시고, 너무 아프시면 바로 말씀해 주세요." → 통증을 관찰하기로 함

**호흡곤란:**
- "숨이 많이 차세요? 지금 산소 수치 확인해 드릴게요." → 호흡곤란이 있는지 확인함
- "산소 마스크 착용해 드릴게요. 더 힘드시면 바로 말씀해 주세요." → 산소흡입 시작함

**출혈:**
- "지금 피가 나오는 상태를 확인할게요. 증상이 심해지면 바로 알려주세요." → 출혈 증상이 있는지 확인함

**신경계 (뇌경색/뇌출혈):**
- "지금 팔다리 힘이 괜찮으세요? 감각이 이상한 부분은 없으세요?" → 신경학적 증상이 있는지 확인함
- "최대한 안정하시고 증상 변화가 있으면 바로 말씀해 주세요." → 사지근력 평가함

### 6-5. 간호진단 처리

- 화면에 표시 안 함
- 간호기록_수정본 D 항목 → 자동으로 팝업에만 추가
- 낙상위험은 항상 포함 🔒

## 7. STEP 5 — 의식/낙상

### A. 의식 확인
💬 "성함이 어떻게 되세요? 오늘 날짜 아세요? 여기가 어딘지 아세요?"

| 선택 | 기록 |
|------|------|
| 의식 명료함 | 의식 명료함 |
| 의식 저하 | 의식수준 : [입력] / Obey command / 지남력 |

### B. 보호자 확인
| 선택 | 기록 |
|------|------|
| 보호자 상주 | 보호자 환자 옆에 상주하고 있음 |
| 보호자 없음 | 보호자 동행하지 않음 |

### C. 낙상교육 스크립트
말할 문장 표시 → 체크 시 간호기록_수정본 P&E 원문 기록

| 📢 말할 문장 | 기록 (원문) |
|-------------|------------|
| "침대에서 내려오실 때는 반드시 간호사를 불러주세요." | 침대에서 내려오실 때 반드시 간호사 호출하도록 교육함 |
| "어지러우시면 바로 앉으시고, 보호자나 담당 간호사에게 알려주세요." | 어지러우면 바로 앉고 보호자나 담당간호사에게 알리도록 교육함 |
| "낙상 위험이 있어서 침대 난간 올려드릴게요." | 낙상 위험성에 대해 설명함 / 침대 난간을 올려줌 / 낙상방지 교육함 |
| (보호자 있을 때) "옆에서 계속 함께 있어 주세요." | 보호자와 같이 있게 함 |
| (보호자 없을 때) "보호자분 연락해 주세요." | 보호자 동반하도록 설명함 |

## 8. 팝업 — 간호기록 - 간호일지

```
간호진술문                      ← 도관 있을 때만
  · [도관 기록]

[증상명] — 간호진술문
  사정 【A&E】 간호진술문        ← 파란색 (수정본 원문만)
  · [A&E 체크 항목들]

  📝 통증 평가 (비고)           ← 회색 (NRS 참고용, 수정본 외 항목)
  · 통증강도 2점 / 통증부위 머리 / 통증양상 쥐어짜듯 / ...

  진단 【D】                    ← 빨간색 (자동)
  · [진단명]

  계획 및 평가 【P&E】 간호진술문 ← 노란색 (수정본 원문만)
  · [체크한 P&E 항목들]

의식/낙상 — 간호진술문
  사정 【A&E】 간호진술문
  · 의식 명료함 / 보호자 상주 등

  진단 【D】
  · 낙상위험 🔒 (항상 자동 포함)

  계획 및 평가 【P&E】 간호진술문
  · [낙상교육 체크 항목들]
```

> **규칙**: 팝업의 A&E·D·P&E 항목은 반드시 간호기록_수정본.xlsx 원문만 사용  
> NRS(통증강도·부위·양상·기간·지속)는 비고란으로만 표시, 정식 진술문 아님
