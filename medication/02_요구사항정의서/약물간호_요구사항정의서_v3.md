## 0. v3 개정 배경 (현장 요구)

**문제:** 간호사들이 약물을 **무엇과 어떻게 섞어서(mix) 어떤 속도로 주는지(administer)** 헷갈려 함.

**원인:** 입고 약물 목록(`소방병원응급실약물사용예정약2606.xlsx`)에는 약품 정보(코드·함량·규격·보험)만 있고 **희석·투여 방법이 없음.** 즉 "무엇을 쓰는가"는 있으나 "어떻게 주는가"가 빠져 있음.

**해결 방향:**
1. 소방병원 실제 입고약물 86개를 앱의 기준 약물 목록으로 확정
2. 약물 클릭 → **희석법 + 투여법이 즉시 한 화면에** 뜨도록 (검색→확인 2탭 이내)
3. 기존 4개 소스에 데이터가 있는 약물은 그대로 카드화, **데이터가 없는 신규 약물은 출처 확보 후 추가** (임의 변형 금지 원칙 유지)

## 5. 탭 1 — 약물 검색 (Drug Reference)

### 5-1. 기능
- 약품명 한글/영문/상품명 실시간 검색 (예: "Norpin"·"노르핀"·"Norepinephrine" 모두 검색)
- 카테고리 필터: 심혈관계 / **진정·진통(RSI)** / 항생제 / 전해질 / 소화기 / 이뇨·수액 / **해독제** / 기타
- 검색 결과 즉시 카드 표시

### 5-2. 약물 카드 구조 (핵심 — "믹스+투여 원클릭")

```
┌─────────────────────────────────────────────┐
│ 💊 Epinephrine HCl          [심혈관계]       │
│    1mg/1ml (대한)                            │
├─────────────────────────────────────────────┤
│  ✅ Bolus 가능 (CPR 시)                      │
│  ⚠️ CIV 시 원액 투여 금지                    │
├─────────────────────────────────────────────┤
│ 📋 믹스(희석) 방법  ← 가장 크게, 최상단      │
│  • Bolus(CPR): 1A 원액 IV push, 4분마다      │
│  • CIV: 1A + N/S 99ml → 10mcg/ml           │
├─────────────────────────────────────────────┤
│ 💉 투여 방법                                 │
│  • CIV: 0.05~2 mcg/kg/min                  │
├─────────────────────────────────────────────┤
│ ⚠️ 주의사항 / 🕐 안정시간 / 🧬 배합금기      │
└─────────────────────────────────────────────┘
```

> **개선점(v3):** 카드에서 "믹스 방법"과 "투여 방법"을 **최상단·최대 글자**로 배치. 간호사가 약 클릭 즉시 이 두 가지를 먼저 보도록.

### 5-3. Bolus 안전 표시 체계 *(v2와 동일)*

| 표시 | 색상 | 의미 |
|------|------|------|
| ✅ Bolus 가능 | 초록 | 정맥 직접 투여 가능 |
| ⚠️ 조건부 가능 | 주황 | CPR 시만 / 고용량 주의 / 권장 안 함 |
| 🚫 Bolus 불가 | 빨강 | 반드시 희석 후 점적 투여 |
| ❌ IV 불가 | 진빨강 | IV 투여 자체 불가 |

### 5-4. 특수 경고 자동 태그 *(v2 + 신규 추가)*

기존: `[Only D5W]` `[Only N/S]` `[단독 투여]` `[non-PVC]` `[12시간 교환]` `[배합금기]` `[Extravasation 주의]` `[냉장보관]` `[flushing 필요]` `[말초/중심 농도 다름]`
신규: `[향정]`(Etomidate·Midazolam 등) `[마약]`(Fentanyl·Pethidine·Morphine) `[반드시 희석]`(KCl) `[근이완—기도확보 필수]`(Rocuronium) `[데이터 확인 중]`(원문 미확보 신규 약물)
