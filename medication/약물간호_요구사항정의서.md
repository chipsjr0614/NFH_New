### 3-3. 약물 카드 구조

```
┌─────────────────────────────────────────┐
│ 💊 Morphine sulfate                      │

├─────────────────────────────────────────┤
│ 📋 희석 방법                             │
│  • 10/30mg: 1A + 50ml (1mg/ml)         │
│  • 100mg:   1A + 100ml (1mg/ml)        │
├─────────────────────────────────────────┤
│ ⏱ 투여 속도                              │
│  • Bolus: 4–10mg, 4–5분 이상            │
│  • CIV: 최대 10mg/hr                   │
├─────────────────────────────────────────┤
│ ⚠️ 주의사항                              │
│  경막외·수막강 내 투여 금지              │
├─────────────────────────────────────────┤
│ 🕐 조제 후 안정시간: 실온 14일          │
└─────────────────────────────────────────┘
```

### 3-4. Bolus 안전 표시

| 표시 | 색상 | 의미 |
|------|------|------|
| ✅ Bolus 가능 | 초록 | 정맥 직접 투여 가능 |
| ⚠️ 조건부 가능 | 주황 | CPR 시만 / 고용량 주의 / 권장 안 함 |
| 🚫 Bolus 불가 | 빨강 | 반드시 희석 후 점적 투여 |
| ❌ IV 불가 | 진빨강 | IV 투여 자체 불가 (Elcatonin) |

### 3-5. 특수 주의 자동 태그

- `[단독 투여]` — 다른 약물과 같은 라인 사용 금지
- `[N/S only]` — D5W 등 다른 수액 사용 금지
- `[말초/중심 농도 다름]` — 접근 경로에 따라 희석 농도 다름
- `[배합금기]` — 특정 약물과 혼합 금지
- `[flushing 필요]` — 투약 전후 N/S 플러싱

## 4. 탭 2 — 지속 주입 계산기 (Drip Calculator)

성인·소아 탭 분리. 체중 입력 → 희석법 + 속도 테이블 자동 계산

### 4-1. 대상 약물 (별도 임상 기준 정의)

**혈관작용 약물 (Vasopressors / Inotropes)**

| 약물 | 농도 예시 | 기본 용량 범위 | 단위 |
|------|---------|-------------|------|
| Epinephrine | 1mg/50ml (20mcg/ml) | 0.01–1 | mcg/kg/min |
| Norepinephrine | 4mg/50ml (80mcg/ml) | 0.01–3 | mcg/kg/min |
| Dopamine | 200mg/50ml (4mg/ml) | 2–20 | mcg/kg/min |
| Dobutamine | 250mg/50ml (5mg/ml) | 2–20 | mcg/kg/min |
| Vasopressin | 20U/50ml (0.4U/ml) | 0.03–0.04 | U/min (고정, 체중 무관) |

**진정·진통**

| 약물 | 농도 예시 | 기본 용량 범위 | 단위 |
|------|---------|-------------|------|
| Midazolam | 50mg/50ml (1mg/ml) | 0.02–0.1 | mg/kg/hr |
| Fentanyl | 500mcg/50ml (10mcg/ml) | 25–200 | mcg/hr |
| Morphine | 10mg/50ml (0.2mg/ml) | 1–10 | mg/hr |
| Ketamine | 500mg/50ml (10mg/ml) | 0.1–0.5 | mg/kg/hr |
| Dexmedetomidine | 200mcg/50ml (4mcg/ml) | 0.2–0.7 | mcg/kg/hr |
| Propofol | 500mg/50ml (10mg/ml) | 5–50 | mcg/kg/min |

**항부정맥**

| 약물 | 농도 예시 | 기본 용량 | 단위 |
|------|---------|---------|------|
| Amiodarone | 150mg/100ml | Loading 150mg/10min → 1mg/min×6hr → 0.5mg/min | mg/min |
| Lidocaine | 1g/250ml (4mg/ml) | 1–4 | mg/min |
| Nicardipine | 10mg/50ml | 5–15 | mg/hr |
| Nitroglycerin | 10mg/50ml (200mcg/ml) | 5–200 | mcg/min |

### 4-2. 계산기 화면 구조

```
[성인] [소아]

약물 선택: [Epinephrine ▼]
체중: [____] kg

─────────────────────────────────────
희석 방법
  Epinephrine 1mg (1A) + N/S 49ml
  → 최종 농도: 20mcg/ml
─────────────────────────────────────
용량 → 주입 속도 테이블 (체중 60kg 기준)

  용량 (mcg/kg/min) │ 속도 (ml/hr)
  ─────────────────────────────
  0.05              │ 9.0
  0.1               │ 18.0
  0.2               │ 36.0
  0.3               │ 54.0
  0.5               │ 90.0  ← 강조
  1.0               │ 180.0
─────────────────────────────────────
⚠️ 임상 메모
  • 소량으로 시작 (0.01 mcg/kg/min)
  • 반드시 중심정맥 투여 권장
  • 5–15분 간격으로 효과 평가 후 조절
```

### 4-3. 소아 Broselow 스타일

```
[소아] 탭 선택

신장(cm) 입력 → 예상 체중 자동 계산 (Broselow 공식)
OR 체중(kg) 직접 입력

Broselow 색상 구간 표시:
  Grey (3–5kg) / Pink (6–7kg) / Red (8–9kg) /
  Purple (10–11kg) / Yellow (12–14kg) / White (15–18kg) /
  Blue (19–23kg) / Orange (24–29kg) / Green (30–36kg)

→ 선택 약물 + 해당 체중 구간 용량 즉시 표시
```
