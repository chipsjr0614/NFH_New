## 4. 탭 1 — 약물 검색 (Drug Reference)

### 4-1. 기능

- 약품명 한글/영문 실시간 검색
- 카테고리 필터: 심혈관계 / 진통·진경 / 항생제 / 전해질 / 소화기 / 기타
- 검색 결과 즉시 카드 표시

### 4-2. 약물 카드 구조

```
┌─────────────────────────────────────────────┐
│ 💊 Epinephrine HCl          [심혈관계]       │
│    1mg/1ml                                  │
├─────────────────────────────────────────────┤
│  ✅ Bolus 가능 (CPR 시)                      │
│  ⚠️ CIV 시 원액 투여 금지                    │
├─────────────────────────────────────────────┤
│ 📋 희석 방법                                 │
│  • Bolus(CPR): 1A 원액 IV push, 4분마다      │
│  • 준CPR: 1A + N/S 9ml → 100mcg/ml          │
│  • CIV: 1A + N/S 99ml → 10mcg/ml           │
│    0.05~2 mcg/kg/min                        │
├─────────────────────────────────────────────┤
│ ⏱ 투여 속도                                  │
│  • CIV: 0.05~2 mcg/kg/min                  │
├─────────────────────────────────────────────┤
│ ⚠️ 주의사항                                  │
│  CPR 아닌 상황에서 원액 투여 금기            │
├─────────────────────────────────────────────┤
│ 🕐 안정시간: 즉시 사용                       │
└─────────────────────────────────────────────┘
```

### 4-3. Bolus 안전 표시 체계

| 표시 | 색상 | 의미 |
|------|------|------|
| ✅ Bolus 가능 | 초록 | 정맥 직접 투여 가능 |
| ⚠️ 조건부 가능 | 주황 | CPR 시만 / 고용량 주의 / 권장 안 함 |
| 🚫 Bolus 불가 | 빨강 | 반드시 희석 후 점적 투여 |
| ❌ IV 불가 | 진빨강 | IV 투여 자체 불가 (Elcatonin — IM only) |

## 5. 탭 2 — 응급 프로토콜 (Emergency Protocols)

응급약물 투약방법.pptx 기반. 상황별 약물 프로토콜을 흐름도 형태로 표시.

### 5-1. 프로토콜 목록

#### ① 서맥 (Bradycardia with Pulse)

```
맥박 있는 서맥 확인
    ↓
증상 있음? (저혈압, 의식저하, 흉통)
    ↓ 예
[Atropine 0.5mg/1ml]
  • 2A(1mg) IV push, 3~5분 간격
  • 최대 3mg(6A)
  • ⚠️ 빠르게 투여 (천천히 주사 시 서맥 악화)
    ↓ 효과 없으면
[Dopamine] CIV → [Epinephrine] CIV
```

#### ② 빈맥 (Tachycardia with Pulse)

```
맥박 있는 빈맥 확인
    ↓
QRS 형태 확인 → 좁은 QRS → PSVT 의심
    ↓
① Valsalva maneuver (미주신경 자극법)
    ↓ 효과 없으면
② Adenosine 투여
```

#### ③ PSVT — Adenosine 투여 프로토콜

| 단계 | 말초정맥 | 중심정맥/심장이식 |
|------|---------|-----------------|
| 1차 | 6mg IV push | 3mg IV push |
| 2차 | 12mg IV push | 6mg IV push |
| 3차 | 12mg IV push | 6mg IV push |

**⚠️ 투여 시 필수 사항 (반드시 준수):**
- 상완 이상 혈관 사용 (심장과 최대한 가깝게)
- 투여 즉시 N/S 20ml flushing + 팔 들기
- 투약 전후 EKG strip 출력
- 의사 Bedside 필수, 응급카트 준비
- 환자 사전 설명: "가슴이 쿵하는 느낌, 심장 멈추는 느낌, 흉통, 호흡곤란 가능"

#### ④ E-cart 약물 빠른 참조

| 약물 | 적응증 | 희석 | 투여 |
|------|--------|------|------|
| Norepinephrine | 저혈압/Shock | 2A + D5W 492ml (16mcg/cc) | 4~12mcg/min 시작 |
| Vasopressin | Septic shock, 요붕증 | 1A + D5W 99ml (0.2U/cc) | 0.01U/min 시작 |
| Esmolol | SVT, 심방세동 | Loading: 100mg+50ml / CIV: 2.5g+240ml | 500mcg/kg bolus → 50~200mcg/kg/min |
| Calcium gluconate | 고칼륨혈증, 부정맥 | 1A + N/S or D5W 100ml | 30분~1시간 (Bolus 금지) |
| Sodium Bicarbonate | Acidosis, 고칼륨혈증 | 원액 | 1mEq/kg IV bolus |
| Magnesium Sulfate | Torsade de pointes | 1~2g + N/S or D5W 10ml | Slow IV 5~20분 |

```

  mcg/min    │  ml/hr

  4          │   15.0  ← 시작 용량
  8          │   30.0
  12         │   45.0
  16         │   60.0
  20         │   75.0
```

## 8. 탭 5 — 소아 Broselow (Pediatric)

**향후 별도 앱으로 개발 예정 (pediatric.html)**  
2025 AHA/AAP PALS 기준 + Broselow 체중 구간 기반

### 8-1. 개발 전제 조건

- 2025 AHA PALS 알고리즘 PDF 원문 직접 확인 필수
- Broselow 색상명 사용 금지 (저작권) → kg 구간으로 대체
- Broselow 1985 원논문 기반 신장→체중 추정 공식 사용

### 8-2. 기능 범위 (기획안)

```
신장(cm) 입력 또는 체중(kg) 직접 입력
    ↓
체중 구간 자동 표시 (3~5kg / 6~7kg / ... / 30~36kg)
    ↓
해당 구간 약물 용량 즉시 표시:
  • 심정지: Epinephrine, Defibrillation J/kg
  • 기도: ETT 크기·깊이, OPA, LMA
  • RSI: Succinylcholine, Rocuronium, Ketamine, Midazolam
  • 쇼크: Fluid bolus, Pressors
  • 경련: Midazolam, Lorazepam
  • 아나필락시스: Epinephrine IM
  • 활력징후 정상 범위
  • 환기: TV, RR, PEEP
```

### 8-3. 포함 약물 (2025 PALS 기준)

| 카테고리 | 약물 |
|---------|------|
| 심정지 | Epinephrine 0.01mg/kg (최대 1mg), Amiodarone 5mg/kg, Lidocaine 1mg/kg |
| 제세동 | 2 J/kg → 4 J/kg → ≥4 J/kg (최대 10 J/kg) |
| RSI 기도 | Succinylcholine 1.5mg/kg, Rocuronium 1.2mg/kg, Ketamine 1~2mg/kg, Midazolam 0.1mg/kg |
| 서맥 | Atropine 0.02mg/kg (최소 0.1mg, 최대 0.5mg) |
| 쇼크 | Fluid bolus 10~20ml/kg 반복, Epinephrine 0.1~1mcg/kg/min |
| 경련 | Midazolam 0.1~0.2mg/kg (IV/IN/IM), Lorazepam 0.1mg/kg |
| 아나필락시스 | Epinephrine 0.01mg/kg IM (최대 0.5mg) |
| 진통 | Morphine 0.1mg/kg, Fentanyl 1mcg/kg |
| 해열진통 | Acetaminophen 15mg/kg, Ibuprofen 10mg/kg |
| 항구토 | Ondansetron 0.15mg/kg |
