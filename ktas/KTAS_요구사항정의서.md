## 2. 전체 10단계 구조

| Step | 제목 | 주요 기능 |
|------|------|-----------|
| **1** | 기본정보 | 날짜 자동, 시간 탭→현재, 환자번호(8자리 제한), 이름, 내원경로 4버튼 |
| **2** | General Impression | 6항목 pill 선택 → SICK/NOT SICK 자동 판정 |
| **3** | ABCD 확인 | A/B/C/D Y/N → D 저하 시 BST modal (건너뛰기 가능) |
| **4** | V/S 입력 | SBP/DBP/HR/RR/BT/SpO2/O2 개별 입력 → 자동 취합 미리보기 |
| **5** | 주호소 선택 | 16개 CC (소형11/중형4/대형1) + 즉시 경고 4종 |
| **6** | Red Flag | CC별 동적 렌더링, 다중 체크박스, 개수 배지 |
| **7** | 세부 질문 | CC별 맞춤 질문 (slider/radio/multi/text) + 추가 증상 태그 |
| **8** | Modifier | V/S+RF+질문 자동 계산 → 예상 KTAS 표시 |
| **9** | 내원동기 & KTAS | 과거력 태그, 수술력 태그, 내원사유/현재호소, 특이사항($), KTAS 내원/변경 |
| **10** | 기록 완성 | 자동 한국어 간호기록 + 항목별 복사 + 전체 엑셀 1줄 복사 |

## 3. Step별 상세 요구사항

---

### Step 1 — 기본정보

#### 날짜 / 시간

| 항목 | 동작 |
|------|------|
| 내원날짜 | 앱 열리면 **오늘 날짜 자동 표시** (`2026-06-06`) |
| 내원시간 | 칸 **탭 한 번 → 현재 시각 즉시 입력** (`14:30`) |
| 접수시간 | 칸 **탭 한 번 → 현재 시각 즉시 입력** |

- 탭 전 `--:--` 표시
- 시계 아이콘 + 라벨 "탭→현재" 표시

#### 환자 정보

| 항목 | 제약 |
|------|------|
| 환자번호 | **최대 8자리** (`maxlength="8"`) |
| 이름 | 텍스트 자유 입력 |

#### 내원경로 버튼 (4개)

```
[ 🚶 도보 ]  [ ♿ 휠체어 ]  [ 🚐 129 사설 ]  [ 🚑 119 구급 ]
```

| 버튼 | State 저장값 |
|------|-------------|
| 도보 | `도보` |
| 휠체어 | `휠체어` |
| 129 사설 | `사설구급차(129)` |
| 119 구급 | `119 구급대` (기본 선택값) |

- 119 구급 → 초록색 하이라이트
- 나머지 → 파란색 하이라이트

---

### Step 2 — General Impression (때깔)

**목적:** 말 걸기 전 0.5초 — "Across the Room" 판단

#### 6개 항목, 각 2~3개 pill 선택

| 항목 | 선택지 | 위험 판정 |
|------|--------|-----------|
| 입원 방식 | 스스로 / 휠체어 / 들것 | 들것 → bad |
| 얼굴색 | 정상 / 창백 / 청색증 / 홍조 | 청색증 → bad |
| 발한 | 없음 / 식은땀 | 식은땀 → bad |
| 호흡 | 편안 / 빠름 / 힘들어 보임 | 힘들어 보임 → bad |
| 의식/반응 | 명료 / 멍함 / 반응 저하 | 반응 저하 → bad |
| 자세 | 편안 / 웅크림 / 움직임 없음 | 움직임 없음 → bad |

#### 자동 판정 박스

- **하나라도 bad** → 빨간 박스 "⚠️ SICK" + "담당의·Charge RN 즉시 알림"
- **모두 ok/warn** → 초록 박스 "✅ NOT Sick"

---

### Step 3 — ABCD 확인

| 항목 | 질문 | 이상 기준 |
|------|------|-----------|
| A — Airway | 말을 할 수 있나요? | 불가 |
| B — Breathing | 스스로 숨을 쉬고 있나요? | 이상 |
| C — Circulation | 맥박? 쇼크 징후? | 이상 |
| D — Disability | 의식이 있나요? | 저하 |

- **하나라도 이상** → "🔴 즉각 소생 필요 (KTAS 1)" 박스 표시
- **D 저하 선택 시** → **BST Modal 자동 팝업**

#### BST Modal

```
혈당 측정 하세요
[혈당 수치 입력] mg/dL
[ 입력 완료 ]
[ 건너뛰기 → ]
```

- 건너뛰기 가능 (필수 아님)
- 입력 시 → Modifier에 저혈당/고혈당 자동 반영

---

### Step 4 — V/S 입력

#### 개별 항목 입력

| 항목 | 단위 | 입력 방식 |
|------|------|-----------|
| SBP (수축기) | mmHg | 숫자 입력 |
| DBP (이완기) | mmHg | 숫자 입력 |
| HR | /min | 숫자 입력 |
| RR | /min | 숫자 입력 |
| BT | °C (소수점 가능) | 숫자 입력 |
| SpO2 | % | 숫자 입력 |
| 산소 투여 | RA / O2 | 버튼 토글 |
| O2 유량 | L/min | O2 선택 시 표시 |

#### 자동 취합 미리보기 (하단)

```
V/S: BP 110/70 / HR 88 / RR 20 / BT 36.5°C / SpO2 98% (RA)
```

산소 투여 시:
```
V/S: BP 110/70 / HR 88 / RR 20 / BT 36.5°C / SpO2 94% (O2 3L)
```

### Step 5 — 주호소 선택 (16개 CC)

#### 레이아웃 구성

**소형 3열 그리드 (11개)**

| CC | 색상 |
|----|------|
| 호흡곤란 | 노랑 |
| 복통 | 주황 |
| 두통 | 주황 |
| 실신/의식변화 | 주황 |
| 발열 | 노랑 |
| 어지럼증 | 노랑 |
| 구토/오심 | 노랑 |
| 설사 | 노랑 |
| 외상 | 초록 |
| 알레르기 | 파랑 |
| 소아 발열 | 초록 |

**중형 2열 그리드 (4개)**

| CC | 색상 | 즉시 경고 |
|----|------|-----------|
| 출혈 (위장관·객혈·코피·질출혈) | 빨강 | — |
| 약물/중독 | 보라 | ✅ 즉시 경고 |
| 신경학적 이상 (CVA 증상) | 파랑 | ✅ 즉시 경고 |
| 정신과 응급 | 보라 | ✅ 즉시 경고 |

**대형 1열 하단 (1개)**

| CC | 색상 | 즉시 경고 |
|----|------|-----------|
| 흉통 — Chest Pain | 빨강 | ✅ 즉시 경고 |

- 기타: 직접 입력 텍스트 필드 제공

#### 즉시 경고 Modal 4종

| CC | 아이콘 | 제목 | 내용 |
|----|--------|------|------|
| 흉통 | ⚡ | ECG 먼저 시행 | 12-lead ECG 즉시, IV, Aspirin 확인 |
| 신경학적 이상 | 🔴 | CVA/Stroke 프로토콜 | Stroke Team, Last Known Well, CT |
| 약물/중독 | ☠️ | 중독/과다복용 확인 | 원인 물질, 기도 확보, 활성탄 |
| 정신과 응급 | 🔒 | 안전 우선 | 자해 도구 제거, 1:1 모니터링 |

| 실신/의식변화 | 4개 |

### Step 7 — 세부 질문

#### CC별 맞춤 질문 구성

| 질문 타입 | 설명 |
|-----------|------|
| `radio` | 단일 선택 버튼 |
| `multi` | 다중 선택 (외상 손상 부위 등) |
| `slider` | NRS 0~10 슬라이더 (색상 그라데이션) |
| `yn` | 있음/아니오 2버튼 |
| `text` | 직접 입력 (Last Known Well 시각 등) |

#### 신경학적 이상 질문 (FAST 기반)

- Last Known Well 시각 (직접 입력)
- 발생 양상: 갑자기 / 서서히
- 편측 위약감 Y/N
- 안면 마비 Y/N
- 언어 장애 Y/N
- 시야 장애 Y/N
- 보행 장애 Y/N
- 두통 NRS 슬라이더

#### 외상 손상 부위 (다중 선택)

두부 / 경추 / 흉부 / 복부 / 골반 / 척추 / 상지 / 하지 / 안면

#### NRS 슬라이더

- 0~10, 실시간 숫자 표시
- 색상: 초록(0) → 노랑(5) → 빨강(10)
- 레이블: 통증 없음 / 경미 / 중등도 / 심함 / 최악

#### 추가 증상 태그

- 직접 입력 후 `+` 버튼 또는 Enter
- 태그로 추가/삭제
- 간호기록에 "기타 동반 증상"으로 자동 포함

### Step 8 — Modifier 자동 계산

#### 자동 계산 규칙

| 조건 | Modifier | 레벨 |
|------|---------|------|
| SBP < 90 | Hemodynamic Instability | HIGH |
| SpO2 < 90% | Critical SpO2 | HIGH |
| SpO2 < 94% | Abnormal SpO2 | HIGH |
| BT ≥ 39.5°C | High Fever | HIGH |
| BT ≥ 38°C | Fever | MID |
| HR ≥ 120 | Tachycardia | HIGH |
| RR ≥ 25 | Tachypnea | HIGH |
| NRS ≥ 7 | Severe Pain | HIGH |
| NRS ≥ 4 | Moderate Pain | MID |
| Red Flag 체크됨 | Red Flag: (항목명) | HIGH |
| 흉통 + 방사통 | High Risk Chest Pain | HIGH |
| 흉통 + 식은땀 | Diaphoresis | HIGH |
| 복통 RLQ + 발열 + 구토 | High Risk Abd. Pain | HIGH |
| 벼락두통 | High Risk Headache — SAH | HIGH |
| 알레르기 + 호흡곤란 | Anaphylaxis | HIGH |
| 신경학적 이상 (무조건) | Neurological Emergency | HIGH |
| 약물/중독 (무조건) | Toxicological Emergency | HIGH |
| 의식 저하 (GI) | Altered Mental Status | HIGH |
| 청색증 (GI) | Cyanosis | HIGH |
| 들것 내원 (GI) | Non-ambulatory | MID |
| BST < 70 | 저혈당 | HIGH |
| BST > 300 | 고혈당 | MID |

#### 예상 KTAS 추천

| 조건 | 추천 |
|------|------|
| HIGH ≥ 1개 | KTAS 2 |
| MID ≥ 2개 | KTAS 3 |
| MID 1개 | KTAS 3 |
| 없음 | KTAS 4 |

※ 최종 판단은 RN이 직접 결정

### Step 9 — 내원동기 & KTAS

#### 과거력 태그 시스템

- 입력 후 `+` 버튼 또는 Enter → `#진단명` 형태 태그 생성
- 예: `#고혈압` `#당뇨` `#COPD`
- 태그 ✕ 버튼으로 삭제

#### 수술력 태그 시스템

- 입력 후 `+` 버튼 → `s/p 수술명` 형태 태그 생성
- 예: `s/p 맹장수술` `s/p 관상동맥스텐트`
- 노란색 태그로 구분

#### 내원사유

- 단일 텍스트 입력
- 예: `복통`, `흉통`, `발열`

#### 현재 호소

- 멀티라인 텍스트 영역
- 예: `6시간 전부터 RLQ 통증 NRS 8/10. 발열·구토 동반.`

#### 특이사항 — $ 마커

- `$` 버튼 클릭 시 텍스트 앞에 자동으로 `$ ` 삽입
- 문제사항·불만·주의 시 사용
- 예: `$ 보호자 폭언 발생`, `$ 자살시도 이력 있음`

#### KTAS 입력

- **내원시** KTAS: 1~5 버튼 (각 KTAS 컬러)
- **변경시** KTAS: 1~5 버튼 (없으면 공란)
- 현재호소 입력 후 하단에 위치

---

### Step 10 — 기록 완성

#### 자동 한국어 간호기록 생성

입력된 모든 정보를 조합해 자연스러운 한국어 문장으로 자동 생성.

**기본 구조:**
```
{과거력} 과거력 있는 환자로, {내원사유}을(를) 주소로 {경로} 내원함.
{CC별 자동 문장}
{현재호소}
기타 동반 증상: {추가 태그}
BST: {혈당값} mg/dL  (D 저하 시)
Red Flag: {항목 나열}
Modifier: {HIGH 항목 나열}
추가: {메모}
```

**CC별 자동 문장 예시:**

| CC | 자동 생성 문장 |
|----|---------------|
| 흉통 | `30분 이내 전부터 중앙 부위 쥐어짜는 양상의 흉통 (NRS 8/10) 발생, 방사통(왼팔)·식은땀 동반.` |
| 복통 | `6시간 전부터 RLQ 부위 찌르는 양상의 복통 (NRS 7/10) 발생, 발열·구토 동반.` |
| 신경학적 이상 | `신경학적 이상 증상으로 내원. Last Known Well: 14:30. FAST 양성 (안면마비·편측위약감).` |

- 생성된 기록은 **수정 가능** (텍스트 영역 직접 편집)

#### 복사 기능

**간호기록 복사 (초록 버튼)**
- 텍스트 영역 전체 복사
- "✅ 복사 완료!" 2초 표시

**항목별 복사 (2×3 그리드)**

| 버튼 | 복사 내용 |
|------|-----------|
| 📅 날짜·시간 | `날짜\t내원시간\t접수시간` |
| 👤 환자정보 | `환자번호\t이름` |
| 🚑 경로·KTAS | `경로\tKTAS내원\tKTAS변경` |
| 🩺 V/S | V/S 전체 문자열 |
| 📋 과거력 | 과거력+수술력 |
| 📝 내원동기 | 내원사유+현재호소 |

**전체 한 줄 복사 → 엑셀 (파란 버튼)**

#### 엑셀 전체 복사 출력 형식

**열 구성 (총 11칸, 탭 구분)**

| 열 | 내용 |
|----|------|
| A | 날짜 |
| B | 내원시간 |
| C | 접수시간 |
| D | 환자번호 |
| E | 이름 |
| F | 내원경로 |
| G | KTAS 내원시 |
| H | KTAS 변경시 |
| I | V/S 전체 |
| **J** | **내원동기 (3줄 → 한 칸)** |
| **K** | **특이사항** |

**J칸 (내원동기) 형식 — 엑셀 셀 내 줄바꿈**

```
- 과거력 : #고혈압 #당뇨 / s/p 맹장수술
- 내원이유 : 복통
- 현재호소 : 6시간 전부터 RLQ 통증 NRS 7/10
```

- 클립보드 형식: 큰따옴표로 감싸고 내부에 `\n` 사용
- 엑셀 붙여넣기 시 한 셀 안에서 줄 나뉨
- 엑셀 "셀 텍스트 줄바꿈" (Alt+H → W) 켜면 모두 보임

## 6. 주요 함수 목록

| 함수 | 역할 |
|------|------|
| `showScreen(n)` | n번 화면으로 이동, 관련 렌더 함수 호출 |
| `setNow('arr'\|'reg')` | 시간 탭 → 현재 시각 입력 |
| `selRoute(el)` | 내원경로 버튼 선택 |
| `selGI(el)` | General Impression pill 선택 + SICK 판정 |
| `selAB(el)` | ABCD Y/N 선택, D 저하 시 BST modal |
| `updateVS()` | V/S 개별 입력 → 취합 미리보기 갱신 |
| `selCC(el)` | 주호소 선택 + 즉시 경고 처리 |
| `renderRF()` | CC에 따른 Red Flag 목록 동적 생성 |
| `renderQ()` | CC에 따른 세부 질문 동적 생성 |
| `calcMod()` | 모든 입력 기반 Modifier 자동 계산 |

| `addSurg()` | 수술력 태그 추가 |
