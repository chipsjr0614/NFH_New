# 트리야지 (초진 / KTAS)

> 응급실 입구에서 담당("초진 간호사")이 **5분 이내**(급하면 1~2분)로 환자를 빠르게 보고 중증도를 분류하는 도구.

**상태:** ✅ **실사용 중** (월요일부터 현장 사용, 큰 문제 없음)

## 흐름
1. 접수 시간(AMR 8자리) + 환자 입력
2. 첫인상 체크(호흡·발화·보행·안색) — 위험하면 즉시 급성(소생실)로
3. 활력징후(혈압·맥박·의식) — 의식 없으면 혈당 먼저(저혈당 거르기)
4. 주호소 + 위험징후 문진(토혈·혈변·임신가능성·통증점수·발병시점 등)
5. 과거력·수술력·투약력
6. KTAS 레벨 판정 → **한 줄 텍스트 복사** → 초진 기록일지에 붙여넣기
