# 요구사항 정의서

**프로젝트명:** 국립소방병원 응급실 간호사 체크리스트 앱  
**작성일:** 2026-05-18  
**요청자:** 국립소방병원 응급실 수간호사  
**버전:** v1.0

---

## 1. 프로젝트 개요

### 1.1 배경
국립소방병원 응급실에 신규 간호사(타 병원 출신 포함)가 다수 근무하고 있어, 응급실 고유의 업무 프로토콜 숙지에 어려움이 있다. 현재 체크리스트가 엑셀 파일로만 존재하여 현장에서 즉시 활용하기 어렵다.

### 1.2 목적
- 응급실 간호사가 환자 상태 변화(입원·퇴원·전원·사망) 및 시술/검사 시 수행해야 할 항목을 모바일·웹에서 즉시 확인
- 신규 간호사의 빠른 적응 지원 (길라잡이 역할)
- 체크리스트 누락으로 인한 의료사고 예방

### 1.3 범위
| 플랫폼 | 대상 기기 | 네트워크 환경 | 업데이트 방식 |
|--------|-----------|--------------|--------------|
| 모바일 앱 | 개인 스마트폰 (iOS/Android) | 인터넷 가능 | 앱 스토어 또는 직접 배포 |
| 웹 | 병원 노트북·데스크탑 | 인터넷 차단 (내부망) | USB 수동 설치 |

### 3.2 체크리스트 조회 (간호사)

#### 3.2.1 카테고리 구성
| 대분류 | 소분류(예시) |
|--------|------------|
| 간호 기록 | 증상(복부 통증, 흉부 통증 등), 통증, 출혈 등 |
| 약물 | 약물명 검색 → 용량·믹스법·속도·주의사항 조회 |
| 필수 업무 | 입원, 퇴원, 전원 |
| 시술 준비 | CT, MRI 등 주요 시술 |
| 업무 가이드 | 업무별 가이드 |

#### 3.2.2 간호 기록 시트 구조
- A열: 대분류 (증상, 통증, 출혈 등)
- B열: 소분류 (복부 통증, 흉부 통증, 통증 관련 사정, 의식 관련 사정 등)
- 이후 열: 간호 과정 + 간호 진술문 (두 항목 상호 연결)
- 화면 동작: "복부 통증" 클릭/검색 시 → 사정·평가 → 간호 진단 → 계획 → 중재 순서로 표시

#### 3.2.3 약물 시트 구조
- **컬럼 구성 (10개):** 분류 / 약물명(영문·코드) / 약물명(한글) / 상품명·별칭(검색 키워드) / 용량 / 용량믹스법 / 투여경로 / 속도·주입시간 / 주의사항 / NFH 수정사항
- **카테고리 18개:** 진통제·해열제 / 마약성 진통제 / 심혈관계 / 항응고제 / 항생제 / 스테로이드 / 이뇨제 / 항구토제 / 기관지확장제 / 진정제·마취제 / 항경련제 / 혈전용해제 / 전해질 / 혈당조절 / 해독제 / 소화기계 / 삼투성이뇨제 / 기타응급약물
- **총 52개 약물 등록 (기본 틀)** — 추후 수간호사가 추가·수정 예정
- **검색 키워드 필드:** 영문명·한글명·상품명·별칭 모두 포함 → 어떤 이름으로 검색해도 해당 약물 조회 가능
- 약물명 검색 또는 클릭 → 해당 약물 상세 정보 즉시 조회

#### 3.2.4 필수 업무 시트 구조
- 입원 / 퇴원 / 전원 각각 단계별(1번, 2번...) 항목 표시
- 클릭 또는 검색으로 즉시 조회 및 체크

#### 3.2.5 체크리스트 화면 동작 (공통)
- 카테고리 선택 → 해당 항목 목록 표시
- 각 항목을 탭/클릭하여 완료 체크 (체크박스 형태)
- 체크 완료 항목은 시각적으로 구분 (색상, 취소선 등)
- 전체 항목 완료 시 완료 표시

#### 3.2.6 CT 촬영 체크리스트 예시 (참고용)
- [ ] 의사 처방 확인
- [ ] 촬영 부위 확인
- [ ] 조영제 사용 여부 확인
- [ ] 조영제 사용 시 GFR(신장 수치) 확인
- [ ] 오른쪽 IV 18G 니들 삽입 여부 확인
- [ ] 환자 금식 여부 확인
- [ ] 동의서 서명 여부 확인

### 3.3 검색 기능
- 검색창에 키워드 입력 시 전체 시트에서 관련 항목 즉시 표시
- 예: "복부 통증" 검색 → 간호 기록 내 사정·평가·진단·계획·중재 노출
- 예: "CT" 검색 → CT 시술 준비 체크리스트 노출
- 예: "약물명" 검색 → 해당 약물 상세 정보 노출

### 3.4 관리자 페이지

#### 3.4.1 접근 권한
- 수간호사, 관리자 계정만 접근 가능
- 로그인(ID/PW) 인증

#### 3.4.2 콘텐츠 관리
- 체크리스트 카테고리 추가 / 수정 / 삭제
- 체크리스트 항목 추가 / 수정 / 삭제 / 순서 변경
- 변경 이력 관리 (누가 언제 수정했는지)

#### 3.4.3 배포 기능 (웹용)
- 최신 콘텐츠를 USB 배포용 파일(패키지)로 내보내기(Export) 기능
- 내보낸 파일을 병원 PC에 설치하면 최신 버전으로 업데이트

### 3.5 콘텐츠 동기화

| 구분 | 방식 |
|------|------|
| 모바일 앱 | 인터넷을 통해 서버에서 최신 콘텐츠 자동 수신 |
| 병원 웹 (PC) | 관리자가 USB로 내보낸 최신 파일을 PC에 수동 설치 |
| 동기화 기준 | 두 플랫폼 모두 동일한 체크리스트 콘텐츠 유지 |

## 4. 비기능 요구사항

### 4.1 사용성
- 신규 간호사도 직관적으로 사용 가능한 UI
- 글씨 크기, 여백, 색상 대비를 고려한 가독성 높은 디자인
- 한 화면에서 카테고리 선택 → 항목 확인까지 최소 탭 수로 접근

### 4.2 오프라인 지원
- 병원 웹(PC): 인터넷 없이 로컬에서 완전히 동작
- 모바일 앱: 최초 1회 다운로드 후 오프라인에서도 기본 조회 가능 (선택 사항)

### 4.3 보안
- 관리자 페이지 로그인 인증 필수
- 일반 간호사는 로그인 없이 바로 사용 (계정 불필요)

### 4.4 성능
- 검색 결과 표시: 입력 후 1초 이내
- 체크리스트 화면 전환: 즉시(1초 이내)

### 4.5 배포 및 설치
- 병원 PC용 웹: USB 하나로 설치 가능한 단일 패키지 제공
- 설치 과정은 비전문가(수간호사)도 수행 가능한 수준

## 5. 플랫폼별 상세 요구사항

### 5.1 모바일 앱
- iOS / Android 모두 지원
- 개인 스마트폰에 설치
- 인터넷 연결 시 콘텐츠 자동 업데이트

- HTML 기반 (브라우저에서 실행)
- 인터넷 없이 로컬 파일로 동작
- USB로 배포되는 단일 패키지 형태
- 노트북·데스크탑 화면 크기에 최적화
