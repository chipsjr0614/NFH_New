# ER메이트

> 응급실 전반의 체크리스트 모음. 세 서비스의 **모태**(원래 전부 여기 넣으려던 것).
> UI가 가장 발전돼 있음.

**상태:** 🟡 **진행 중** — 세팅·UI는 잡힘. **마스터 데이터 업데이트**와 **체크리스트 "이유" 추가**가 남음.

## 구성 영역
- **검사/시술 준비 체크리스트** — 영상검사(CT·MRI 등)
  - CT: 조영/비조영 분기 → 조영이면 주사바늘(손목 위 20G↑)·동의서·GFR 60↑·조영제 알러지·전처치·임신여부 / 비조영은 임신여부만
  - MRI: 금속 제거 핵심(보청기·목걸이·속옷·심전도 스티커 등)
- **혈액검사 체크리스트** (추가 예정)
- **필수 업무** — 입원 / 퇴원 / 전원(타 병원) / 사망
- **업무 가이드** — 자주 하지만 놓치는 업무(예: 수혈 — ABO·꼬리표·IV 20G↑·알러지·시간제한)

> 참고: **약물 계산기**는 독립 서비스로 분리됨 → [`../약물계산기`](../약물계산기)
