## 1. 핵심 발견 — 데이터는 4가지 성격으로 갈린다

전부 체크박스로 만들면 "수액 속도표"나 "GCS 점수표"까지 체크박스가 되는 문제가 생깁니다. 그래서 **성격별로 화면을 다르게** 줍니다.

| # | 성격 | 간호사가 원하는 것 | 화면 방식 |
|---|------|------------------|----------|
| ① | **준비물형** | "뭐 챙기지? 빠뜨리면 안 되는데" | **분기 체크리스트 + 이송 게이트** |
| ② | **절차형** | "순서대로 따라가면 끝나게" | **단계 타임라인** (1→2→3) |

## 5. 분기 질문 상세 (★ 표시 항목)

> 아래 분기는 **엑셀에 글로 적힌 조건**을 옮긴 것입니다. "이 항목이 정확히 어느 분기에 속하는가"는 **검수 필요**.

### CT — 조영/비조영
- **조영:** 동의서 / ⚠️GFR 60↑ / 20G↑ 손목 위 IV / 조영제 알러지 전처치 / 귀금속 제거·환의
- **비조영:** 임신 여부 · 귀금속 제거만 (위 4개 숨김)

### MRI — Sedation 여부 (+ 금속 항상 확인)
- **공통:** ⚠️금속물질(안경·보청기·틀니·ECG lead·pacemaker 등) 제거 직접 확인 / GFR 60↑
- **Sedation:** midazolam·O2·ambu·이송모니터 + 의료진 동반 / 검사후 의식·호흡 확인

### CAG — ICU 여부 + 기존 항혈소판제
- **ICU행:** 입실동의서 추가
- **기존 복용 중:** Aspirin·clopidogrel 각 1tab만 (주치의 확인) / 미복용: DAPT 로딩 4종

### 수혈 — 3중 분기 + 용량표(참조)
- **혈액형 등록?** 있음=ABO 1개 / 없음=2개
- **혈액제제?** RBC(15분 50~75→150~200, 4h내) / FFP(30분~1h) / PLT / Cryo …
- **과거 부작용?** 있음 → ⚠️전처치(Chlorpheniramine+NS 50ml IVS 후 시작)

### 전원 — 이송수단 + 정규/비정규
- **사설구급차:** 비용 안내·연락 / **정규:** 전원소견서=전원전담간호사
- **공통:** 전원동의서·의무기록/CD·⚠️퇴실30분전 V/S·외래예약

### 파상풍 — 5년 이내 접종?
- **없음:** 파상풍 약(냉장) 투약 / 2개면 양팔 따로
- **있음:** ✅ 접종 불필요

*(그 외 ★분기: PCN·흉수천자 좌우확인, Echo TTE/TEE, DJstent 설치/교환/제거, C-line·Pacemaker 신규/AST, 홀터 정규/비정규, 요추천자 위치 — 표 참조)*
