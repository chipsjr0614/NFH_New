# 간호기록 (NFH) ★ 최우선

> 환자가 구역(관찰/급성)에 들어오면 담당 간호사가 깊게 사정(assessment)하고,
> 그 결과로 **정형화된 간호기록을 자동 생성**하는 도구. 스무고개식 문진이 핵심.

**상태:** 🔴 **재설계 중** — 기존 "사정 툴 + 간호기록 툴"을 합치다 망가져 버그 多.
**처음부터 다시 만들기로 결정.** 현 단계는 요구사항 역정리.

## 목표 흐름
1. **환자 확인 팝업** (성함·팔찌 확인, 담당 소개, 금식 안내 — 멘트 대본 제공)
2. **도관 확인** (IV/주사바늘·콧줄·소변줄 — 타 병원에서 달고 올 수 있음)
3. **주호소 → 증상 카테고리 문진** (예: 복통 → 구역/구토·설사·열감·혈변·임신가능성·통증[부위·양상·점수·발병] 등 딱 필요한 10항목, 채팅형)
4. **사정 → 간호진단(예: 급성 통증) → 평가·중재**
5. **낙상 예방 교육**(디폴트, 누락 시 처벌 대상)
6. 예외: "약만 맞으러 온" 환자는 *이상 없음* 선택 시 건너뛰기

## 핵심 데이터 — "7만 항목" 정형 진술문
- 병원에서 실제 쓰는 정형 문구(예: "토했음"❌ → "오심 구토 있음"✅).
- 응급실+공통만 카테고리화하면 실제 문항은 **100~200개 이하**.
- ⚠️ **마스터 데이터가 아직 저장소에 없음(칩스 보유)** — 재구축의 최대 블로커.
