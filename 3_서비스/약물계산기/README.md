# 약물 계산기

> 체중 입력 시 약물 용량을 자동 계산하는 도구. 특히 **소아(PALS)** 는 체중별 용량이 많아 핵심.
> (원래 ER메이트의 한 도구였으나, 내용 분량이 많아 **독립 서비스**로 분리)

**상태:** 🟢 **계산기 완성** — "집어넣기만 하면 됨" 수준. 확인 절차·배포 경로만 확정 필요.

## 범위
- 체중 기반 용량 계산 (소아 PALS 포함)
- 자주 쓰는 주요 약물 검색·조회 (용량·믹스법·속도·주의사항)
- 항생제 병용 투여 참고
